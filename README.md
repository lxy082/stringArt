# 绕线画（String Art）

这是一个纯前端的绕线画生成器项目，基于 HTML/CSS/JavaScript 与 Canvas，可在浏览器中直接运行。

## 运行方式（本地或 GitHub Codespaces）

在仓库根目录执行：

```bash
python -m http.server 8000
```

然后在浏览器中打开：

```
http://localhost:8000/StringArtGenerator.html
```

## 测试（GitHub Codespaces）

当前项目暂无自动化测试脚本。

在 GitHub Codespaces 中可使用以下命令进行页面加载的手动冒烟检查：

```bash
python -m http.server 8000
```

启动后访问：

```
http://localhost:8000/StringArtGenerator.html
```
